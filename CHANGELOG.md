### RÉUNION 19/02/2026 - Changement du nom du projet

Il faut chercher un nouveau nom pour le projet.

Le nom actuel _"Koncord"_ a plusieurs problèmes :
- Il commence par un _"K"_, ce qui fait trop penser à la suite KDE
- Il est nécessaire de sortir du marché de l'alternative

Nous avons proposé :
- _"Babili"_ : traduction de _"chatter"_ en Espéranto, ne convient pas (trop mignon et ressemble à _"Babel"_)
- _"Reto"_ : traduction de _"réseau"_ en Espéranto, intéressant mais nom de domaine trop cher
- _"Atrio"_ : dérivé du latin "atrium", intéressant mais nom de domaine trop cher

Le prix du nom de domaine n'est pas immédiatement problématique (nous pouvons utiliser les github pages), mais pourrait poser problème dans le futur.

**Aucune décision définitive, mais _"reto"_ semble plaire à SL, et MD.**
**En attente d'une prochaine réunion pour clarifier les nouvelles décisions avec WT.**
